研究背景

核心研究问题

结果不平等

人们最终获得收入的高低

例：有人年收入5万，有人年收入50万

机会不平等

获得收入的机会是否平等

例：家庭背景、户籍、性别等因素在多大程度上决定了你的收入？

核心问题

"寒门难出贵子" —— 家庭背景在多大程度上决定了个人命运？
传统参数方法可能低估机会不平等，需要机器学习新方法

理论框架

Roemer的机会不平等理论

收入分解模型（公式1）

Y = f(Circumstances) + g(Effort)

Circumstances（环境因素）：个人无法选择的因素  |  Effort（努力因素）：个人可以控制的因素

Circumstances 环境因素

  • 父亲/母亲教育年限
  • 父亲/母亲政治面貌
  • 父亲/母亲职业层级
  • 家庭人均收入（对数）
  • 户籍类型（城市/农村）
  • 性别、年龄、民族

Effort 努力因素

  • 个人教育投资
  • 工作努力程度
  • 职业选择
  • 风险承担意愿
  • 时间分配

理论框架

IACE指数定义

IACE（Inequality of Opportunity Index）

IACE = I(YC) / I(Y)

其中 YC = E(Income | Circumstances) 为由环境因素预测的收入

取值范围

0 = 完全机会平等

1 = 完全机会不平等

不平等指标

Gini系数 和 MLD

本文主要采用Gini系数

研究方法

机器学习预测模型

回归树模型（公式2）

f(x) = Σm=1M cm·I(x ∈ Rm)

cm为子空间Rm的输出值

条件推断树（公式4-6）

H0: D(Y|Xj) = D(Y)
padj = 1 - (1-p)m

Bonferroni校正消除选择偏误

随机森林（公式8-9）

μ̂(x) = Σi=1n ωi(x)Yi
ωi(x) = (1/k) Σt=1k ωi(x,θt)

权重平均提升稳健性

收入条件期望（公式3）

E(Income | Circumstances) = Rm = ĉm = avg(yi | xi ∈ Rm)

研究方法

SHAP因素分解方法

SHAP（Shapley Additive Explanations）

基于博弈论的合作博弈分配方法，将IACE指数分解到各个环境因素

每个因素的贡献度 = 该因素对预测收入的边际影响

分解步骤

  1. 计算总体不平等Gini
  2. 计算个体不平等贡献度
  3. 使用SHAP获得因素贡献度
  4. 剔除特定因素后重新计算
  5. 比较Gini变动

结论

机器学习方法测算结果明显高于传统线性模型，证实了"函数形式简化容易低估机会不平等"的理论推断
原因：除非引入大量交互项和高阶项，否则线性模型容易将环境因素影响遗漏在随机扰动项中

研究结论

核心发现

结论一

IACE指数为0.244~0.307，贡献总体不平等的38%~52%，明显高于传统方法

结论二

城乡间机会不平等占比最高（40%-57%），户籍制度是最大来源

结论三

父亲特征持续拉大收入差距，母亲特征缓解不平等

结论四

分布不平等远高于均值不平等，收入风险不平等约为均值的2倍

政策建议

缩小机会不平等的政策路径

前端治理

  • 推动城乡公共服务均等化
  • 加大农村教育投入
  • 缩小城乡师资差距

反歧视政策

  • 消除就业性别歧视
  • 保障女性职场权益
  • 建立反歧视法律体系

监测体系

  • 将分布不平等纳入统计监测
  • 关注收入风险和增长空间
  • 建立大数据监测平台

预期管理

  • 建立收入分配预警系统
  • 针对性治理干预
  • 消除长期负面情绪

研究贡献

本文创新点

方法创新

  • 首次将机器学习应用于中国IACE测算
  • 突破传统线性模型的函数形式假设
  • 使用SHAP方法实现非线性分解
  • 提供更准确的估计结果

内容创新

  • 从均值不平等拓展到分布不平等
  • 引入分位数回归森林方法
  • 揭示收入风险差距的重要性
  • 丰富机会不平等的外延

研究展望

局限与未来方向

研究局限

  • Circumstance变量选择仍有主观性
  • Effort变量难以精确测量
  • CGSS调查间隔较长（5年）
  • 因果推断存在挑战

未来方向

  • 追踪研究：代际流动性变化
  • 国际比较研究
  • 政策评估：改革效果检验
  • 微观数据监测系统建设
